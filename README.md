# Запуск
В директории api:  
1. `cp .env.example .env`  
2. `npm i (опционально)`  
3. `npm run compose:up ` 
4. `npm run db:sync`  
5. `npm run dev`

В директории client:
1. `npm run i`
2. `npm run dev`

Пока умники не сделали нормальные миграции - на сайте нужно регистрировать юзеров самостоятельно
